aitrade
=================================
A simple trading system for AI

## Install

```bash
pip3 install -r requirements.txt
```

## 配置

    1. 复制config.example.yaml为config.yaml
    2. 根据文件注释修改内容

## 创建虚拟环境

```bash
python3 -m venv venv

# 激活虚拟环境
# Windows系统
venv\Scripts\activate
# Linux/Mac系统
source venv/bin/activate

# 退出虚拟环境
# Windows系统
venv\Scripts\deactivate
# Linux/Mac系统
source venv/bin/deactivate
 ```

## 运行

```bash
python -m aitrade
```
